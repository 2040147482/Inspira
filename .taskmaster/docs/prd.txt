# 产品需求文档（PRD）：IdeaBox 灵感火花工具箱

## 1. 产品概述

**产品名称**：IdeaBox 灵感火花工具箱
 **产品定位**：为内容创作者、品牌从业者、营销人和自由撰稿人提供一站式创意生成工具，涵盖标题生成、品牌命名、广告语生成、创意提示和灵感收藏。
 **目标语言**：支持中英文双语切换
 **平台形态**：Web 应用，适配桌面与移动端

------

## 2. 背景与机会

当前市面上的创意生成工具存在以下不足：

- 多为单一功能，缺乏一体化整合体验；
- 英文导向，中文支持不足；
- 灵感管理与创意复用机制缺失。

IdeaBox 致力于打造一个轻量、高效、支持中英文切换的创意平台，为创作者带来真正可用的内容灵感辅助。

------

## 3. 技术栈选型

### 前端

- **框架**：Next.js（支持 SSR、动态路由、App Router）
- **UI 框架**：Tailwind CSS（快速构建响应式设计）
- **组件库推荐**：shadcn/ui（适配 Tailwind 的现代组件）

### 后端服务

- **Supabase**：集成用户认证、数据库（PostgreSQL）、文件存储和 Realtime 功能
  - 用户登录与权限管理
  - 收藏夹、灵感图谱、灵感日记等核心数据存储
  - 评论系统支持实时互动

### AI 模型接入

- **OpenAI API**：用于标题生成、改写器、灵感助手、图像提示生成
  - GPT-4 / GPT-3.5：文本类 AI 功能
  - DALL·E（可选）：图像生成能力

### 部署平台

- **Vercel**：推荐部署平台，适配 Next.js 与 Serverless API

### 其他工具建议

- **国际化**：`next-i18next` 或 `next-intl` 支持中英切换
- **状态管理**：轻量使用 React Context 或 Zustand
- **监控与日志**：Vercel Analytics + Supabase Logs

## 4. 目标用户画像

| 用户类型      | 特征描述                                                 |
| ------------- | -------------------------------------------------------- |
| 内容创作者    | 公众号、小红书、视频号作者，需要吸睛标题与话题点子       |
| 品牌方/创业者 | 需要快速生成品牌名、广告语、口号等商业创意内容           |
| 写作者/撰稿人 | 寻求灵感激发和结构化创作提示                             |

### 5.1 标题生成器（Title Generator）

#### 功能定位
根据关键主题与目标平台，利用 AI 生成多风格、可直接发布的爆款标题，并提供评分与改写入口。

#### 核心功能点
- **关键词输入**：支持手动输入或从当前灵感/剪贴板一键导入  
- **平台场景选择**：小红书 / B站 / 抖音 / YouTube / 公众号 / 博客  
- **风格预设**：权威、搞笑、悬念、情绪钩子、数字列表、问题式  
- **智能变量**：自动提取热词、数字、情绪词以增强吸引力  
- **批量生成**：一次输出 5～10 条候选标题，支持刷新变体  
- **AI 评分**：点击单条标题可查看吸引力、清晰度、平台适配度评分  
- **一键改写**：将候选标题发送至「创意改写器」继续深度打磨  
- **收藏 & 分组**：勾选后存入「灵感收藏夹」指定分组  
- **多语言**：中英文互译与双语并排展示

#### 页面结构预览（草图示意）
```
+---------------------------------------------------+
| 🎯 爆款标题生成器                 [历史记录 🔽]    |
+---------------------------------------------------+
| 关键词/主题： [_________________________]        |
| 目标平台：  [小红书 ▼]   风格： [悬念 ▼]         |
| ▶️ [生成 10 个标题]                               |
+---------------------------------------------------+
| 1. 标题候选文本 A      [评分 87]  [改写] [收藏]   |
| 2. 标题候选文本 B      [评分 79]  [改写] [收藏]   |
| ...                                                   |
+---------------------------------------------------+
```

#### 用户流程（简图）
```mermaid
flowchart TD
  A[输入关键词 / 选择模板] --> B[选择平台与风格]
  B --> C[点击生成]
  C --> D[查看候选标题及评分]
  D --> E{用户动作}
  E --> F[收藏]
  E --> G[发送至改写器]
  E --> H[刷新获取新变体]
```

### 5.2 品牌命名器（Brand Name Generator）

#### 功能定位
基于产品描述与命名风格快速生成具有差异化和品牌记忆点的多语言品牌名称，并提供域名可用性初步检查。

#### 核心功能点
- **产品描述输入**：支持简短描述或长文本输入
- **命名风格标签**：高端、国潮、国际化、极简、科技、可爱等
- **语言选项**：中文 / 英文 / 混合
- **智能拆词组合**：AI 按语义拆分关键词并重新组合
- **相似品牌冲突检测**：提示已知商标/品牌相似度
- **域名初检**：查询 .com/.cn/.io 等主要后缀可注册性（调用第三方 API）
- **批量生成**：一次输出 10+ 备选名，附简要含义说明
- **收藏 & 标星**：可将心仪名称加入收藏夹
- **一键导出**：CSV/Markdown

  B --> C[生成品牌名]

### 5.3 广告语生成器（Slogan Generator）

#### 功能定位
为品牌或活动快速生成多种长度、风格的广告口号，帮助强化传播记忆点与情感共鸣。

#### 核心功能点
- **品牌/卖点输入**：一句核心利益点或品牌理念
- **语气与节奏**：简洁、押韵、情感、号召、幽默
- **字数限制**：6 字 / 10 字 / 15~20 字 可选
- **变体模式**：平铺直叙、排比、反转问句、押头韵
- **情绪热词库**：自动引入高情绪分词增强冲击力
- **批量输出**：一次生成 8~12 条 slogan
- **评分与标注**：高亮关键词并给出情感/记忆度评分
- **收藏 & A/B**：可选两条加入标题优化器做 A/B 对比
- **中英互译**：双语口号对照

### 5.4 随机创意提示器（Random Idea Prompt Tool）

#### 功能定位
通过"一键摇骰子"随机组合主题、情境与创作角度，激活灵感并打破惯性思维。

#### 核心功能点
- **骰子按钮**：点击即生成全新 Prompt
- **组合维度**：场景 + 目标受众 + 切入角度 + 形式
- **关键词模式**：输入词后随机延展变体
- **锁定维度**：可固定某一要素，对其他要素重新随机
- **生成历史**：最近 20 次记录可回看
- **多语言**：中/英切换
- **收藏 & 发送**：可直接收藏或发送至对应生成器

#### 页面结构预览（草图示意）

+---------------------------------------------------+

| 角度：反直觉调研  形式：信息图                    |

### 5.5 灵感收藏夹（Inspiration Library）

#### 功能定位
集中管理所有收藏的标题、品牌名、口号及社区灵感，支持多维标签和版本追溯，成为个人创意资产库。

- **分组文件夹**：项目 / 类型 / 标签 三层筛选
- **批量操作**：移动、删除、导出、合并

### 5.6 灵感广场（Inspiration Plaza）

#### 功能定位
打造开放且友好的灵感分享社区，用户可发布创意、求助与反馈，并通过互动提升灵感质量与曝光度。

#### 核心功能点
- **发布灵感贴**：标题+正文+关联标签+所用工具
- **互动机制**：点赞、评论、@提醒、收藏
- **热度算法**：结合点赞数、评论数、收藏数、发布时间
- **筛选排序**：最新 / 热门 / 未解决 / 标签
- **悬赏求助**：可设置虚拟积分奖励优质回答
- **精选板块**：运营团队每日 Pick 高质量内容
- **快速引用**：一键将他人灵感复制到自己收藏夹
- **违规检测**：AI 风控过滤不当内容
- **多语言切换**：自动翻译评论

### 5.7 图像提示词生成器（Prompt Generator for AI Art）

- #### 功能定位：

  辅助用户为 AI 图像生成工具（如 Midjourney、DALL·E、文心一格、Stable Diffusion 等）高效生成结构化提示词（Prompt），以获得更精准的图像输出结果。支持创作者从简单意图出发，构建专业且具视觉指导性的 Prompt。

  #### 核心功能点：

  - **输入方式**：
    - 简洁描述输入（如：一个穿越沙漠的机器人）
    - 或选择"可视引导模式"：通过选择风格/画面要素自动构建 Prompt
  - **参数选项**：
    - 主体描述（角色、场景、物品、动作）
    - 风格（插画 / 写实 / 像素风 / 漫画 / 科幻 / 油画 / 摄影等）
    - 镜头语言（特写 / 广角 / 俯视 / 仰视 / 鸟瞰 / 景深虚化）
    - 构图风格（对称 / 动态构图 / 留白等）
    - 画幅比例（1:1、16:9、3:4 等）
    - 画面氛围（阳光 / 黑暗 / 魔幻 / 梦幻 / 忧郁）
    - 灯光与质感（体积光 / 柔光 / 反射 / 油润）
    - 附加关键词（"trending on Artstation"、"unreal engine 5"等）
    - 支持中英文切换
  - **输出格式**：
    - 自动拼接结构良好的 Prompt（中英文可切换）
    - 支持 Midjourney、DALL·E、SD 风格定制
    - 可复制、收藏、导出
  - **预设模板**（一键填充结构）：
    - 人物头像（角色+风格+表情+镜头）
    - 产品展示图（场景+材质+背景）
    - 科幻场景（设定+环境+科技细节）

### 5.8 AI 灵感助手（AI Inspiration Assistant）

#### 功能定位：

帮助用户对已有的内容（如标题、广告语、文章开头）进行多风格、多版本改写，适配不同平台风格或语境。

#### 核心功能点：

- 输入一段文字内容（如一句标题或段落）
- 可选择重写方向：
  - 更吸引人
  - 更专业权威
  - 更幽默 / 更口语化
  - 更简洁有力
- 可选择目标平台语气（小红书、知乎、YouTube、广告文案等）
- 支持中英文输入和输出
- 支持"一稿多写"展示 3~5 个风格改写版本
- 可将结果复制、收藏、或一键发给 AI 灵感助手继续加工

#### 页面结构预览（草图示意）：

```
+--------------------------------------------+
| 输入框：请输入想要改写的文字内容         |
|（支持中文 / 英文）                        |
+--------------------------------------------+
| 改写方向选择：[吸引人] [幽默] [专业] ...   |
| 平台语气选择：[小红书] [知乎] [YouTube]    |
+--------------------------------------------+
| ▶️ [开始改写]                              |
+--------------------------------------------+
| 结果展示：                                 |
| 1. 版本一（幽默风）                        |
| 2. 版本二（专业风）                        |
| 3. 版本三（极简风）                        |
|    [复制] [收藏] [发给灵感助手继续修改]     |
+--------------------------------------------+
```

#### 用户流程（简图）：

```
flowchart TD
  A[输入原始文字内容] --> B[选择改写方向与语气]
  B --> C[点击开始改写]
  C --> D[展示多个风格版本]
  D --> E{用户操作}
  E --> F[复制 / 收藏]

### 5.9 创意改写器（Content Rewriter）

### 5.10 标题优化器 / A/B 测试建议（Title Optimizer）

#### 功能定位：

帮助用户对多个标题版本进行对比分析，提供优化建议、评分与改写建议，适用于标题写作的优化迭代及 A/B 测试前期准备。

#### 核心功能点：

- 多标题输入对比：用户可同时输入多个标题版本（如 2~5 个）
- AI 评分维度（每项支持 0-100 分）：
  - 吸引力（是否抓眼球）
  - 清晰度（表达是否简明清楚）
  - 创意性（是否有新意、出人意料）
  - 平台适配度（是否适合目标平台，如小红书 / B 站）
- 自动标注强项与弱项词汇（如热词、长尾词、冗词）
- 提供整体建议与每个标题的重写推荐
- 一键导出对比报告（图表 + 文字分析）
- 可跳转至标题生成器或改写器继续创作

#### 页面结构草图：

```
[输入框组：输入 2~5 个标题候选版本]
🎯 目标平台选择：[小红书] [知乎] [B站]
▶️ [开始分析]

分析结果：
- 标题 1：评分 85（吸引力：90 / 清晰度：78）
- 标题 2：评分 74（吸引力：80 / 创意性：60）
⚠️ 弱项：用词略显陈旧，建议更换动词
✅ 建议改写：[试试这个：这 3 个涨粉方法，最后一个惊呆我了]
[复制] [收藏] [转跳至改写器]
📊 报告导出：[下载 PDF]
```

#### 用户流程：

```mermaid
flowchart TD
  A[用户输入多个标题版本] --> B[选择目标平台]
  B --> C[点击开始分析]
  C --> D[系统返回评分和点评]
  D --> E{用户操作}
  E --> F[接受建议 / 改写]
  E --> G[跳转改写器或标题生成器]
  E --> H[导出报告或复制收藏]
```

#### 增强计划：

- 允许上传已有标题表现数据（如 CTR、浏览量），进行数据辅助评分
- 集成与灵感广场的互动：可发起"社区投票版 A/B 测试"
- 针对不同语言、文化背景提供不同优化建议（如中文语境更注重情绪钩子）

### 5.11 灵感图谱（Inspiration Mindmap）

#### 功能定位
通过可视化「脑图」把散落的灵感、生成内容与收藏资源串联成结构化创意网络，帮助用户从"点子"迅速扩展到"完整方案 / 脚本大纲"。

#### 核心功能点
| 模块 | 说明 |
| --- | --- |
| **中心节点** | 输入主题词或拖入收藏内容即可生成脑图中心 |
| **AI 自动延展** | 一键生成 3-6 个一级分支，任意节点可再次 AI 延展 |
| **手动编辑** | 拖拽、双击编辑文本，快捷键增/删/连结节点 |
| **节点类型** | 颜色 / 图标区分：标题、段落、脚本、图片、CTA 等 |
| **多语言** | 支持中英双语显示，一键翻译节点文本 |
| **工具联动** | 节点菜单可发送到标题生成器、改写器或发布至灵感广场 |
| **保存与版本** | 多项目云端保存，支持快照版本对比与恢复 |
| **导出** | PNG / PDF / Markdown / JSON |

#### 页面结构预览（线框稿）
```
+---------------------------------------------------+
| 🧠 灵感图谱 |  新建图谱🔽 | 导出 | 保存 |
+---------------------------------------------------+
|  [中心主题输入：请输入关键词]   [生成]            |
+---------------------------------------------------+
|         （可缩放 / 拖拽的脑图区）                |
|   ● 时间管理技巧                                 |
|   ├─○ 爆款标题灵感 (5)                           |
|   ├─○ 短视频脚本骨架 (3)                         |
|   └─○ 配图风格建议 (4)                           |
+---------------------------------------------------+
| ➕ 添加节点 | 🤖 AI 延展 | 🔄 撤销/重做 | 🗑️ 删除 |
+---------------------------------------------------+
```

#### 用户流程
```mermaid
flowchart TD
  A[创建 / 打开灵感图谱] --> B[输入或拖入中心灵感]
  B --> C[AI 生成一级分支]
  C --> D[用户手动编辑 / 再次 AI 延展]
  D --> E{节点操作}
  E --> F[发送到其他工具]
  E --> G[保存 / 导出]
```

#### 典型应用场景
| 场景 | 示例 |
| --- | --- |
| 视频选题策划 | "数字游民生活" → 输出脚本结构、标题清单、镜头语言 |
| 品牌活动脑暴 | "环保主题 Campaign" → 输出受众洞察、口号、传播渠道 |
| 写作提纲 | "元宇宙科普" → 输出章节标题、案例素材、金句 |
| 课程设计 | "时间管理" → 输出课时目录、互动练习、故事 |

#### 增强计划
1. **实时协作**：多人同时编辑、评论，光标同步  
2. **智能聚类**：相似节点自动归类并推荐标签  
3. **热度指示**：根据收藏 / 引用次数显示节点热度环形条  
4. **反向生成**：上传脑图 → AI 生成文章大纲或 PPT 框架

### 5.12 灵感时间线 / 创意日记（Inspiration Timeline & Diary）

#### 功能定位
将用户每日生成、收藏及社区互动的灵感按时间轴自动汇聚，形成「创意成长日记」。方便复盘创作过程、追踪灵感来源，并激励持续创作。

#### 核心功能点
| 模块 | 说明 |
| --- | --- |
| **时间轴视图** | 以日/周/月为单位展示所有生成记录、收藏与社区发帖 |
| **事件分类** | 图标区分：生成(⚡)、收藏(⭐)、评论(💬)、发布(🚀) |
| **快速回溯** | 点击事件可查看原始内容，并可再次发送至工具编辑 |
| **搜索 & 过滤** | 按关键词、工具类型、日期范围筛选灵感事件 |
| **成长统计** | 每日/周生成条数、收藏次数、评论互动等数据仪表盘 |
| **日记笔记** | 用户可为某日添加心得文字或上传图片，支持 Markdown |
| **提醒与激励** | 支持每日创作提醒推送，展示连续创作天数徽章 |
| **隐私设置** | 私密 / 公开 / 好友可见三档可选
